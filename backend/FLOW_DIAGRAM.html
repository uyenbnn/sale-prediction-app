Model Tuning Process

Cross Validation

  • TimeSeriesSplit (5 folds)
  • Date-sorted training data

Models Tuned

  • Ridge: alpha grid
  • Random Forest: trees, depth, leaf size

Search Method

  • GridSearchCV per model
  • Scoring: negative MSE
  • n_jobs=1 (stable on Windows)

Selection Output

  • Compute MAE and RMSE
  • Select lowest RMSE
  • Save model + metrics.json

Request Flows (Simple)

POST /api/predict

  • Build one feature row
  • Predict sales
  • Return suggestion

POST /api/forecast-range

  • Input: start_date + end_date
  • Loop dates and predict daily
  • Return daily rows + summary

GET /api/model-evaluation

  • Check cache
  • Evaluate full dataset
  • Generate chart and metrics

POST /api/compare-products

  • Predict each product over 7 days
  • Aggregate and rank demand
